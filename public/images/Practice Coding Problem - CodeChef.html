Ballon d Or

Leo has already won the Ballon d'Or 888 times, so he is really impressed with it (is he?).

Leo has an array AAA containing NNN integers. Each element of this array is either 111 or 222.
He wants to figure out if the product of all the elements of the array can be written as an 888-th power of some integer, i.e, k8k^8k8 for some integer kkk.

Print Yes if it can and No if it can't.

Input Format

  • The first line of input will contain a single integer TTT, denoting the number of test cases.
  • Each test case consists of two lines of input.
    • The first line of each test case contains a single integer NNN the number of integers in Leo's array.
    • The second line contains NNN space-separated integers A1,A2,,ANA_1, A_2, \ldots, A_NA1,A2,,AN, denoting the elements of array AAA.

Output Format

For each test case, output on a new line "YES" if a valid kkk exists, and "No" otherwise (without quotes).

Each letter of the output may be printed in either uppercase or lowercase, i.e, the strings no, No, nO, and NO will all be treated as equivalent.

Constraints

  • 1T1051 \leq T \leq 10^51T105
  • 1N21051 \leq N \leq 2\cdot 10^51N2105
  • 1Ai21 \leq A_i \leq 21Ai2
  • The sum of NNN over all test cases won't exceed 21052\cdot 10^52105.

Sample 1:

Input
Output
2
1
2
9
2 2 2 2 1 2 2 2 2
NO
YES

Explanation:

Test case 111: The product of the array is 222, which can't be written as k8k^8k8 for any integer kkk. So, the answer is No.

Test case 222: The product of the array is 222212222=2562\times 2\times 2\times 2\times 1\times 2\times 2\times 2\times 2 = 256222212222=256, which equals 282^828. So, the answer is Yes.
